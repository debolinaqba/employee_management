# Employee Management System

A simple command-line Employee Management System built using **Python** and **Object-Oriented Programming (OOP)**.

## Features

- Add Employee
- Display Employees
- Search Employee
- Delete Employee
- Calculate Annual Salary

## Technologies

- Python 3
- Docker
- Git & GitHub


## Run the Project

```bash
pip install -r requirements.txt
python main.py
```

## Run with Docker

```bash
docker build -t employee-app .
docker run -it employee-app
```
